# Contributing

Thanks for your interest in contributing to the uwo-shared-platform.

Guidelines:

- Please open issues to discuss larger changes before submitting pull requests.
- Follow Conventional Commits for commit messages (e.g. feat:, fix:, chore:).
- Keep changes focused and add tests where appropriate.
- Run linting and type checks before opening a PR.

Pull request process:

1. Create a branch from main for your change.
2. Open a pull request describing the change and link any relevant issues.
3. Request a review from the code owners listed in CODEOWNERS.

Thank you!
